Broken Image Handler

What are we creating?

A visual & creative solution for handling broken images.

What is the purpose of this?

To override the browser default ugly image replacement with a more pleasing visual design which will enhance the user experience of your website/webapp

When would I use this?

In websites/webapps/projects as a backup solution for any broken images/links that might be misplaced on the server.

Instructions: Whether it's a typo in the img tag source, or the image itself was misplaced or removed on the server - images break all the time on the web. The problem is, the browsers default way to display a broken image is really ugly. You can fix that and actually use it as an opportunity to enhance the user's experience on your web pages with CSS!
